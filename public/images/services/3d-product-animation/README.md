# 3D Product Animation

**Category:** Service Images
**Purpose:** Showcase 3D product animation work

## Usage in Cards:
- Display 1-2 key images in service card
- Rotate through animations in gallery
- Show product lifecycle or transformation

## File Guidelines:
- **Recommended Size:** 1200x800px
- **Format:** WebP or JPG
- **File Size:** 80-150KB per image
- **Quantity:** 2-4 images recommended

## Example Naming:
- `01-product-render.webp`
- `02-animation-frame.webp`
- `03-detail-shot.webp`

## Component Integration:
```tsx
import Image from 'next/image'

<Image
  src="/images/services/3d-product-animation/01-product-render.webp"
  alt="3D Product Animation"
  width={1200}
  height={800}
/>
```

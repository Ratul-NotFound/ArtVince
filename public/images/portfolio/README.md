# Portfolio Cases

**Category:** Portfolio Images
**Purpose:** Display complete project case studies

## Usage:
- Portfolio/projects page
- Client showcase
- Work samples gallery

## File Guidelines:
- **Recommended Size:** 1200x800px or 1600x900px
- **Format:** WebP (recommended)
- **File Size:** 120-250KB per image
- **Quantity:** 2-5 images per project

## Naming Convention:
- `[project-name]-[type].webp`
- Examples: 
  - `game-project-01-hero.webp`
  - `game-project-01-detail.webp`
  - `animation-project-01-hero.webp`

## Portfolio Structure:
```
project-01/
  ├── hero-image.webp
  ├── detail-01.webp
  └── detail-02.webp

project-02/
  ├── hero-image.webp
  └── gallery.webp
```

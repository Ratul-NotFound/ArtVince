# Branding Assets

**Category:** Branding
**Purpose:** Store logos, favicons, and brand assets

## Files to Include:

### Logos:
- `logo-light.png` - Logo on dark background
- `logo-dark.png` - Logo on light background
- `logo-icon.png` - Icon-only version

### Favicon:
- `favicon.ico` - Browser favicon

### Social Media:
- `og-image.png` - Open Graph image (1200x630px)
- `twitter-image.png` - Twitter card image

### Brand Guidelines:
- Logo files at multiple sizes
- Color palette reference
- Typography samples

## File Guidelines:
- **Logo:** PNG with transparency
- **Favicon:** ICO format
- **Social:** JPG/PNG, recommended sizes below

## Recommended Sizes:
- Logo: Original size, 1000x1000px+
- Logo Icon: 256x256px
- Favicon: 32x32px (multiple formats)
- OG Image: 1200x630px
- Twitter: 1200x675px

## Usage:
```tsx
// In layout.tsx or next.config.js
<link rel="icon" href="/images/branding/favicon.ico" />
<meta property="og:image" content="/images/branding/og-image.png" />
```

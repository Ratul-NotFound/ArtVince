# Hero Section Images

**Category:** Hero Images
**Purpose:** Display hero carousel and large showcase images

## Usage:
- Hero section main carousel
- Background images
- Large showcase displays

## File Guidelines:
- **Recommended Size:** 1920x1080px minimum
- **Format:** WebP (recommended)
- **File Size:** 150-300KB per image
- **Quantity:** 3-6 carousel images

## Naming Convention:
- `hero-[description].webp`
- Examples: `hero-main.webp`, `hero-carousel-01.webp`

## Component Integration:
```tsx
const heroImages = [
  "/images/hero/hero-carousel-01.webp",
  "/images/hero/hero-carousel-02.webp",
  "/images/hero/hero-carousel-03.webp"
]
```

## Responsive Sizes:
- Desktop: 1920x1080px
- Tablet: 1024x576px
- Mobile: 640x360px (use srcset or Next.js Image)

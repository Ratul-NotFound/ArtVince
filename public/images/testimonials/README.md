# Testimonials & Client Logos

**Category:** Testimonial Images
**Purpose:** Display client logos and testimonial visuals

## Usage:
- Testimonials section
- Client logos carousel
- Trust indicators

## File Guidelines:
- **Recommended Size:** 150x150px (square)
- **Format:** PNG (recommended for logos) or WebP
- **File Size:** 5-30KB per image
- **Quantity:** 3-6 client logos recommended

## Naming Convention:
- `[company-name]-logo.png`
- Examples: `netflix-logo.png`, `microsoft-logo.png`

## Component Integration:
```tsx
<Image
  src="/images/testimonials/client-logo.png"
  alt="Client Name"
  width={150}
  height={150}
/>
```
